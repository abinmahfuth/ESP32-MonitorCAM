### Notes on version 2:
- Change pin configuration for to ESP-CAM AI-thinker
- ST7735 connector pin correction
- Change camera FCP connector to include both top and bottom pins.
- Added a power plane.
- Reduce data traces to 6mil to keep impedence at 50 ohms.
